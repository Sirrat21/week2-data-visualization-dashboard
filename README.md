# Dashboard Summary
## Project Title

Interactive Data Visualization Dashboard using Streamlit

## Objective

The objective of this project is to analyze the Netflix Movies and TV Shows dataset and present meaningful insights through an interactive dashboard developed using Streamlit. The dashboard allows users to explore the dataset using filters and visualizations.

## Dataset

* **Dataset Name:** Netflix Movies and TV Shows Dataset
* **Total Records:** 8,800+ records
* **Source:** Kaggle

## Data Preparation

The following preprocessing steps were performed:

* Removed duplicate records.
* Handled missing values.
* Converted the `date_added` column to datetime format.
* Prepared the dataset for visualization.

## Dashboard Features

* Project Introduction
* Dataset Overview
* Dataset Preview
* Interactive Sidebar Filters
* KPI Cards
* Multiple Interactive Charts
* Correlation Heatmap
* Insights and Conclusion
* Download Filtered Dataset

## Visualizations Included

* Bar Chart
* Pie Chart
* Line Chart
* Histogram
* Box Plot
* Scatter Plot
* Correlation Heatmap

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Streamlit

## Key Insights

* Movies make up a larger portion of Netflix's catalog than TV Shows.
* Netflix content increased significantly after 2015.
* A few countries contribute most of the available content.
* Interactive filters help users explore the dataset efficiently.

## Conclusion

This project demonstrates practical skills in data cleaning, exploratory data analysis, interactive visualization, and dashboard development using Streamlit. The dashboard provides an effective way to analyze and communicate insights from the Netflix dataset.
